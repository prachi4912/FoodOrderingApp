# Food Ordering System (Java OOP Console Application)
## Project Description

This is a Java Console Based Application developed using Object Oriented Programming (OOP) concepts.

The system allows users to order food items such as Pizza and Burger through a console interface. The application demonstrates core Java concepts like classes, objects, inheritance, encapsulation, and polymorphism.
## Features

- Order different food items
- Display menu items
- Calculate total price
- Demonstrates Java OOP concepts
- Simple console-based interface
- ## Future Improvements

- Add more food items
- Create GUI using Java Swing
- Add database using MySQL
